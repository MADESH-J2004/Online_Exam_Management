# Online_Exam_Management
AI-powered online examination system built using FastAPI, React, and MySQL with role-based dashboards for students and teachers, automated MCQ generation using LLM, real-time face detection proctoring, and student performance prediction.
# Online Exam Platform

## 🚀 Overview
An AI-powered online examination system designed to automate exam creation, monitoring, and evaluation using modern web technologies and machine learning.

## 🔥 Features
- Role-based dashboards for students and teachers  
- Automated MCQ generation using LLM (Llama3 via Ollama)  
- Real-time face detection proctoring (face-api.js)  
- Student performance prediction using Logistic Regression  
- Secure backend with FastAPI and MySQL  

## 🛠 Tech Stack
Python, FastAPI, React, MySQL, Machine Learning, face-api.js, Ollama (LLM)

## 📌 Outcome
Improved exam automation, reduced manual effort, and enabled intelligent monitoring and performance analysis.
